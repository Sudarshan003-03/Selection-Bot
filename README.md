# Selection-Bot
Interactive Tata Motors Commercial Vehicle Spare Parts Chatbot built with HTML, CSS, and JavaScript. Combines selection-based and conversational chat to help users search genuine spare parts by vehicle type, BS standard, model, and part name, displaying pricing, warranty, and compatibility details in a responsive interface.
